#h5-projects
> 针对h5搭建一个多项目应用的基础配置框架,并封装一些工具函数

## 开发环境和生产环境配置

### build目录
1. webpack配置有三个部分：基本配置，开发环境和生产环境
2. 基本配置配置路口和通用的module和plugin
3. 开发环境需要配置webpack-dev-server，热重载和错误提示和其他
4. 生产环境需要压缩js,提取css，优化html和开启gzip压缩

### config目录
1. 配置同上
